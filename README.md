# SnakeGame
贪吃蛇小游戏，类minecraft风格，简单支持排行榜系统，无聊捣鼓出来的，部分用了AI修改代码，侵删

## 后端
### Express
代码在分支[express](https://github.com/ccpopy/SnakeGame/tree/express)上，需要node环境
### 生成对应的WASM文件和js文件
代码在分支[c](https://github.com/ccpopy/SnakeGame/tree/c)上
